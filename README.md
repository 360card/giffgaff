Giffgaff 的政策允许邮寄到大陆的，不过仍有大陆的粉丝反馈遇到了Country（国家）停留在英国无法选择的情况。这种情况下一般更换网络环境都能解决，比如更换到手机移动网络下操作，或者走一个其它国家的代理。

有下边几个关键点需要注意：

#### 1 First name 和Last name

First name 这里填写拼音的名， Last name 填写拼音的姓，建议全部大写，提高辨识度。

#### 2 Email

请准确填写，这样我们就能收到物流寄出的反馈，粉丝实测过国内常用的邮箱也是能正确接收到邮件反馈的，极少数邮箱会收到垃圾邮件中（像139.com）。

#### 3 Country

如果是邮寄到中国大陆这里选择China，如果是邮寄到港、澳、台等其它地区根据实际情况准确填写。

#### 4 Address line 1

如果是邮寄到中国大陆，大部分情形直接使用汉语拼音从区/县一级按字英文空格分割（参考上图），包括空格在内少于50 个字符长度，为保证简洁不需要填写街道，直接写学校名称、大厦名称，楼号、单元号和房间号统一用<数字楼号>-<数字单元号>-<数字房间号> 表述，如果仍然写不完，可以补充到Address line 2 ( optional) 部分。拼音可以通过[在线汉字转拼音](https://www.qqxiuzi.cn/zh/pinyin/) 转换。

> 部分大学对国际信件的地址格式有要求，须为标准英文格式，请以学校的要求为准。

#### 5 Address line 2 (optional)

如果是邮寄到大陆，务必在地址末尾加空格并附上你的手机联系方式，其它地区也可以附上以增加送达率。

#### 6 Town/City

如果是邮寄到大陆，填写你所在的城市的拼音，如果是邮寄到港、澳、台等其它地区根据实际情况准确填写。

#### 7 State/Province

如果是邮寄到大陆，填写你所在省或省级城市的拼音大写，如果是邮寄到港、澳、台等其它地区根据实际情况准确填写省或州信息。

#### 8 Postcode

一定要准确填写，这决定了你的邮件到达中国后能否到达的关键。比如在中国大陆，不太了解自己区域邮编的可以参考[大陆邮政系统邮编查询](https://www.spb.gov.cn/search5_gy/html/searchResult.html?column=%E5%85%A8%E9%83%A8&searchWord=%E9%82%AE%E6%94%BF%E7%BC%96%E7%A0%81&columnId=0&siteCode=bm71000001)。

> 据称国际平信在国内的丢失率达 30% 多。所以把上面的填写的信息保存下来，后面重复提交时还会用到。

### 保号

giffgaff 官方的政策是每半年有余额变动或者充值。 giffgaff 官方会在保号周期到期前一周发送提醒邮件，还请特别留意。

在中国大陆发短信是最简便的方法。总结一下，有以下几种方式保号：

* 进行一次充值  
* 拨打一个电话（不能是giffgaff客服、紧急电话等免费电话）  
* 发送一条短信（不能是giffgaff客服免费短信）  
* 使用数据连一次互联网

这里推荐发给英国（其它运营商）的手机客服热线（+44 797 300 0186），发送号码填写0044 7973000186，内容任意。

### 配置短信中心号码

在一些极端情况下（比如从实体卡转换到eSIM 等），giffgaff 的短信收发有异常，这种情况下需要设置giffgaff 的短信中心号码。

* 进入“设置”界面后，找到“短信”或“消息”等相关选项。
* 在“短信”或“消息”界面中，找到“短信中心”或“消息中心”等相关选项。
* 点击“短信中心”或“消息中心”选项，进入编辑短信中心号码的界面。
* 输入giffgaff的短信中心号码：+447802092035，保存设置。
